Active projects

Pan & Tilt camera tracking

Pan & Tilt servo system using raspberry pi and face classification to track faces real-time. Instructables & source code to be posted soon.

Open source low budget 3D printer

3D printer using majority 3D printed parts and low cost hardware. Designed for simplicity. V2 in production.


Programming projects

A* Maze solver

Python project for visually demostrating A* Maze solving with custom mazes

image-fetcher

Published pypi module for large image dataset gathering implementing multithreading. Particually useful for machine-learning and with in depth documentation

Automated Dog Classifier

Complete machine learning project with automated image fetching, keras backend and flask web app frontend. Designed as a generalised multi-class ML project.

Django app

Django web application with django rest backend application and automated testing. Good example project for those building django apps.

2D android game (pokemon)

Pokemon style android game built without standard engine. For uni project. A nice tool for developing any generic 2D games with simple screens, sprites, etc.

Arduino / Leap motion library

Optimised arduino serial read/writing and leap motion integration. Was used to create real time hand reading to relay to inmoov robotic hand (video in repository)

JavaFX interactive map

JavaFx virtual browser with external control to calling js functions. Ideal for any interactive demo projects using a browser such as interactive maps using external controls (sensors) such as leap motion

Java xbox controller

Source code and libraries for xbox controller integration into java. Was implemented for farmbot testing. Good for most robotics projects

ML chatbot

Simple flask python app interface using chatbot (bot-famework) interface. USed on AI camp. Excellent for creating a chatbot with AI backend

Stuck Overflow

Keras Machine Learning project from Kainos AI Camp hackathon. Uses text analysis to identify key libraries/functions from stack overflow answers and maps to questions
